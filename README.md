# 📝 AlphaToDoApp

SwiftUI ve Firebase kullanılarak geliştirilmiş modern bir görev (To-Do) uygulamasıdır.

---

## 🚀 Özellikler

- 🔐 Firebase Authentication ile kullanıcı girişi
- ☁️ Cloud Firestore ile veri kaydetme
- ➕ Görev ekleme
- 🗑️ Görev silme
- 📋 Gerçek zamanlı görev listesi
- 👤 Profil ekranı

---

## 🛠️ Kullanılan Teknolojiler

- SwiftUI
- Firebase Auth
- Cloud Firestore
- MVVM Architecture

---
